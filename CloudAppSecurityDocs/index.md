# <a name="cloud-app-security-documentation"></a>Документация по Cloud App Security

Администрирование и защита электронной почты, документов и конфиденциальных данных, используемых совместно за пределами вашей компании. С помощью Azure Information Protection вы можете усилить защиту данных (от простой классификации до внедренных меток и разрешений), независимо от времени, расположения и предоставленного доступа.
