# <a name="troubleshooting-microsoft--cloud-app-security-status"></a>Устранение неполадок с состоянием Microsoft Cloud App Security

Теперь вы можете проверить текущее состояние службы Microsoft Cloud App Security, перейдя по адресу [status.cloudappsecurity.com](https://status.cloudappsecurity.com), или непосредственно на портале, выбрав **Справка**>**Состояние системы**.
